# skew_symmetric_error_disb_in_FS_via_CobbDouglas
This repository contains the equations, model summary, and an R implementation sketch used in the thesis "Skew-Symmetric Error Distribution Analysis of Elasticity Dynamics of Firm Financial Statement Components: A Case Study in Indonesian Stock Markets using The Cobb–Douglas Linearized Log Form" (presented at ICITEE 2025). See the conference poster/presentation for full details. The paper is the author's undergraduate thesis for his final year in college.

## 1. Motivation

Standard Cobb–Douglas regressions assume **Gaussian residuals**, an assumption often violated in firm-level financial statement data due to:
- firm heterogeneity,
- asymmetric growth shocks,
- accounting nonlinearities,
- market frictions in emerging economies.

This thesis replaces the normal error assumption with a **skew-symmetric distribution**, allowing:
- asymmetric residual behavior,
- improved likelihood fit,
- better inference on elasticities.

## 2. Model
Let firm revenue be generated by total assets and labor:

$$R = \kappa A^{\beta_1} N^{\beta_2} \varepsilon$$

Taking logs:

$$\log R_i = \beta_0 + \beta_1 \log A_i + \beta_2 \log N_i + \epsilon_i$$

where:
- $R_i$: total revenue  
- $A_i$: total assets  
- $N_i$: number of employees  
- $\beta_1, \beta_2$: output elasticities
The formulation follows the classic Cobb-Douglas production function, only the assets and revenue as well as number of employees constitute capital, output, and labor $K, Y,L$ respectively
---

## 3. Skew-Symmetric Error Specification

Instead of assuming:

$$\epsilon \sim \mathcal{N}(0, \sigma^2)$$

we assume a **skew-normal distribution**:

$$f(\epsilon;\zeta,\omega,\alpha) = \frac{2}{\omega} \phi\left(\frac{\epsilon-\zeta}{\omega}\right) \Phi\left(\alpha\frac{\epsilon-\zeta}{\omega}\right)$$

where:
- $\zeta$: location parameter  
- $\omega > 0$: scale  
- $\alpha$: skewness parameter  
- $\phi(\cdot)$: standard normal PDF  
- $\Phi(\cdot)$: standard normal CDF  



---

## 4. Log-Likelihood Function

Let residuals be:

$$\epsilon_i = \log R_i - (\beta_0 + \beta_1 \log A_i + \beta_2 \log N_i)$$

Define:

$$z_i = \frac{\epsilon_i - \zeta}{\omega}$$

The log-likelihood per observation is:

$$\log f(\epsilon_i) = \log 2 - \log \omega + \log \phi(z_i) + \log \Phi(\alpha z_i)$$

Total log-likelihood:

$$\mathcal{L}(\theta) = \sum_{i=1}^n \Big[ \log 2 - \log \omega + \log \phi(z_i) + \log \Phi(\alpha z_i) \Big]$$

with parameter vector:

$$\theta = (\beta_0, \beta_1, \beta_2, \zeta, \omega, \alpha)$$

This allows residuals to be **asymmetric around zero**, which is empirically relevant for firm data. The residuals here produce a unique log likelihood plot

## 5. Estimation Method: BFGS

Parameters are estimated via **Maximum Likelihood** using the **BFGS quasi-Newton algorithm**.

Given:
- parameter update $s_k = \theta_{k+1} - \theta_k$
- gradient difference $y_k = \nabla \mathcal{L}_{k+1} - \nabla \mathcal{L}_k$

The inverse Hessian approximation updates as:

$$H_{k+1} = (I - \rho_k s_k y_k^\top) H_k (I - \rho_k y_k s_k^\top) + \rho_k s_k s_k^\top$$

where:

$$\rho_k = \frac{1}{y_k^\top s_k}$$

---

## 6. Economic Interpretation

### Marginal Products

$$\frac{\partial R}{\partial A} = \beta_1 \frac{R}{A}, \quad \frac{\partial R}{\partial N} = \beta_2 \frac{R}{N}$$

### Marginal Rate of Technical Substitution (MRTS)

$$\text{MRTS}_{A,N} = \frac{\beta_1}{\beta_2} \cdot \frac{N}{A}$$

### Returns to Scale

- **Constant:** $\beta_1 + \beta_2 = 1$  
- **Increasing:** $\beta_1 + \beta_2 > 1$  
- **Decreasing:** $\beta_1 + \beta_2 < 1$

The economic interpertation showed by the plot produces the classic Cobb-Douglas trade-off between Capital and Labor, via employee expenses
